### Installation

Requirements:
* [ripgrep](https://github.com/BurntSushi/ripgrep#installation);

Clone this repo:

```sh
git clone https://github.com/nvim-lua/kickstart.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

Then run:

```sh
nvim --headless "+Lazy! sync" +qa
```
